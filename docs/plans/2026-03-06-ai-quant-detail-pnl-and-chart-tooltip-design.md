# AI量化策略详情收益额与曲线交互增强设计

## 1. 背景
- 当前策略详情页已有：收益率、最大回撤、胜率、交易次数。
- 当前收益曲线仅展示线条，不支持 hover 查看细节。

## 2. 目标（已确认）
1. 新增“总收益额”展示。
2. “总收益额”卡片中以小字显示“今日收益（+/-）”。
3. 收益曲线支持 hover：显示“时间点 + 相邻点变化百分比（+/-）”。

## 3. 信息展示设计
### 3.1 指标区
- 新增一张指标卡：`总收益额`
- 卡片内容：
  - 主值（大字）：总收益额（金额）
  - 副值（小字）：今日收益（+/- 金额）

### 3.2 曲线 Tooltip
- 鼠标悬停某个点时展示：
  - 时间点（该点 ts）
  - 相邻点变化百分比（+/-）
- 首点没有上一点，显示：`变化: --`

## 4. 数据口径
- 总收益额：`当前净值 - 初始净值`
- 今日收益：`今日最后净值 - 今日第一净值`
- 相邻点变化百分比：`(当前点 - 上一点) / 上一点 * 100%`
- 所有百分比保留 2 位小数；正值带 `+`。

## 5. 交互细节
- 曲线 hover 时显示：
  - 当前点高亮圆点
  - 垂直参考线
  - tooltip 跟随鼠标并自动避让边界
- tooltip 靠右溢出时翻转到左侧显示。

## 6. 验收标准
1. 策略详情页出现“总收益额（主值）+ 今日收益（副值）”卡片。
2. 收益曲线 hover 可显示时间点与相邻点变化百分比。
3. 第一个点显示 `变化: --`。
4. 正负号展示正确，数值保留两位小数。
5. 无后端场景下可稳定展示（基于本地 mock 数据）。

## 7. 边界
- 本轮不接后端接口。
- 本轮不改策略业务流程，仅增强详情展示。
