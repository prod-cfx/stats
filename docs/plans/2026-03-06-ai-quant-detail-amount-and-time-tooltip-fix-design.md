# AI量化策略详情金额口径与时间提示修正设计

## 1. 问题背景
- 当前详情页收益曲线 tooltip 仍出现 `T7` 等占位时间，未满足“对应真实时间点”要求。
- 当前“总收益额”展示口径不符合预期，应展示“总额（本金+累计盈亏）”，并在下方显示“今日盈亏”。

## 2. 目标（已确认）
1. 使用 mock 数据修正总收益额口径。
2. 总收益额卡片展示：
   - 主值：总额（例如 `10020 USDT`）
   - 小字：今日收益（例如 `今日 +20 USDT`）
3. 曲线 tooltip 显示：
   - 完整时间 `YYYY-MM-DD HH:mm`
   - 相邻点变化百分比（+/-）

## 3. 数据与口径
### 3.1 策略金额字段
- 新增（或补齐）字段：`initialCapital = 10000`（mock）
- 总额计算：`initialCapital + 累计盈亏`
- 今日收益：`今日最后净值 - 今日第一净值`

### 3.2 Tooltip 显示
- 时间：`YYYY-MM-DD HH:mm`
- 变化：`(当前点 - 上一点) / 上一点 * 100%`
- 首点显示：`变化: --`

## 4. 旧数据兼容迁移
- 在读取 `ai_quant_strategy_store_v1` 时自动执行迁移：
  1. 若 `equitySeries.ts` 为占位格式（如 `T7`），重建为真实时间序列。
  2. 若缺少 `initialCapital`，补 `10000`。
- 迁移后自动回写 localStorage，用户无需手动清缓存。

## 5. 验收标准
1. 详情页“总收益额”主值显示总额（非净变化值）。
2. 卡片小字显示“今日 +/- 金额”。
3. 曲线 tooltip 时间为完整格式 `YYYY-MM-DD HH:mm`。
4. tooltip 显示相邻点变化百分比（+/-）。
5. 历史本地数据自动迁移后不再出现 `T7`。
