# AI 量化 MVP 设计（Brainstorming 输出）

日期：2026-03-04

## 1. 目标与范围

### 1.1 目标用户
- 小白用户：通过自然语言描述交易想法，系统引导补齐参数，生成可回测、可部署的量化策略。

### 1.2 MVP 目标
- 跑通一条完整主链路：
  - 用户自然语言输入
  - AI 识别并引导补参
  - 生成策略配置
  - 回测结果展示
  - 用户确认后一键部署实盘

### 1.3 核心原则
- 自然语言不要求完全命中固定句式。
- 以“策略模板 + 参数可调”替代“自由脚本直出”。
- 优先稳定、可解释、可审计。

## 2. 方案选择

### 2.1 选型结论
采用方案 A：`模板槽位匹配 + 参数补全对话 + 结构化策略执行`。

### 2.2 不采用方案
- 不采用“LLM 直接生成执行脚本”作为 MVP 主路径：可控性、稳定性和审计能力不足。

## 3. 产品形态（MVP）

### 3.1 策略库
- 首个模板：`价格动量策略（可调参数版）`
- 能力：
  - 自然语言预填参数
  - 参数面板可编辑
  - 可回测
  - 可部署

### 3.2 前端核心页面
- 策略广场：展示模板入口。
- 自然语言对话页：对话补参 + 参数确认。
- 回测结果页/面板：收益、回撤、胜率、交易次数、曲线、明细。
- 个人中心：我的策略、运行状态、订单/执行日志。

### 3.3 一键部署
- MVP 要求支持真实下单（非仅模拟执行）。

## 4. 架构设计

### 4.1 组件分层
1. NLU 层：抽取意图与参数槽位。
2. 模板匹配层：命中 `价格动量策略（可调参数版）`。
3. 对话补参层：逐项追问缺失参数（一次一个）。
4. 参数编译层：生成统一策略配置（结构化 JSON）。
5. 回测层：基于历史行情执行回测。
6. 部署执行层：校验账号后部署实盘并运行。
7. 监控审计层：记录策略、回测、部署、订单和错误日志。

### 4.2 对话与参数面板协同
- 自然语言负责“选模板 + 预填参数”。
- 参数面板允许用户手动调整所有参数。
- 对话修改与面板修改双向同步，始终以最新参数快照为准。

## 5. 参数模型

### 5.1 必填参数（不完整不可回测）
- 交易所（MVP 支持 Binance、OKX）
- 交易对（多币种）
- 入场条件（窗口 + 阈值）
- 出场条件（窗口 + 阈值）
- 单笔资金比例

### 5.2 可选高级参数（可缺省）
- 手续费率
- 滑点率
- 冷却时间
- 最小成交额
- 最大并发持仓数
- 其他执行层参数

### 5.3 缺省策略
- 高级参数未回答时允许继续回测/部署。
- 系统自动填默认值，并在确认卡中明确标注“使用默认值”的字段。

## 6. 数据流与状态机

### 6.1 主链路
1. 用户输入自然语言。
2. NLU 预填可识别参数。
3. 命中策略模板。
4. 对话引擎逐项追问缺失必填项。
5. 用户在参数面板可随时修改。
6. 参数完整后执行回测。
7. 用户确认后按同一参数快照一键部署。

### 6.2 策略实例状态
- `DRAFT`：创建/补参中
- `READY_TO_BACKTEST`：必填参数齐全
- `BACKTESTED`：已产出回测结果
- `DEPLOY_PENDING`：部署中
- `RUNNING`：实盘运行
- `PAUSED`：暂停
- `ERROR`：异常
- `STOPPED`：停止

## 7. 错误处理与安全边界

### 7.1 参数与语义错误
- 交易所-市场-交易对不匹配：阻断并提示修正。
- 百分比、窗口、仓位超范围：字段级报错。
- 解析置信度低：转确认式提问，不强猜。

### 7.2 回测错误
- 行情拉取失败：允许保留参数重试。
- 样本不足或零成交：返回调参建议。

### 7.3 部署与实盘错误
- API Key 无效/权限不足/IP 限制：明确失败原因。
- 下单失败：有限重试 + 幂等键，防重复下单。
- 异常状态可追踪并可恢复（`PAUSED/ERROR`）。

### 7.4 最小风控基线
- 仓位与可用余额校验。
- 防重复触发/重复下单。
- 部署前二次确认（展示最终参数快照与默认值项）。

## 8. 验收标准（MVP）

满足以下 6 条视为通过：
1. 不完整自然语言可进入补参流程并命中模板。
2. 必填参数可通过对话逐项补齐。
3. 用户可在面板自由调整交易所、币种、时间和百分比及高级参数。
4. 使用真实历史行情完成回测并输出核心指标与明细。
5. 用户确认后可完成真实部署并进入运行态。
6. 全链路状态与错误可审计可追踪。

## 9. 非目标（本期不做）
- 多策略编排与策略组合优化。
- 大规模模板市场和复杂推荐系统。
- 自动化参数寻优与遗传算法。
- 复杂跨交易所套利策略。
